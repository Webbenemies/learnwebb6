A list is a collection of items, and it can take different forms.

HTML provides various ways to create lists on a webpage. There are 3 main types of lists in HTML

html list

  1. ordered lists
  2. unordered lists
  3. and Description lists

These lists provide a way to structure and organize information on a webpage. They can be used individually or nested within each other to create more complex structures.

Ordered lists [ <ol> ]

Ordered lists are used when the order of the items is important. They are created using the <ol> (ordered list) element, and each item in the list is defined with the <li> element.

Example

You can also use different types of numbering or lettering by using the type attribute.

This type overrides the one used by its parent <ol> element, if any.

Common values include:

  • A: uppercase letters
  • a: lowercase letters
  • I: uppercase Roman numerals
  • i: lowercase Roman numerals
  • 1: numbers

Unordered List [ <ol> ]

Unordered lists in HTML are created using the <ul> (unordered list) element. Each item in the list is defined using the <li> (list item) element. Unordered lists are often used when the order of items is not significant, and each item is typically marked with a bullet point.

Description list

Description lists <dl>, <dt>, <dd> in HTML are used to create a list of terms and their corresponding descriptions.

Example

The <dl> element defines the description list

The <dt> element is used to define the term (or name)

The <dd> element is used to define the description.

These elements don't have any specific attributes, you can use CSS to style them for better presentation. For example, you can use styles to adjust the font size, color, spacing, and other visual aspects of the description list.
