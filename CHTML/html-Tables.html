fundamentals of html table

A table is a structured set of data organized in rows and columns. Tables are commonly used to display information in a tabular form, making it easier for users to understand and compare data. HTML provides specific elements for creating tables, including <table>, <th>, <tr>, and <td>.

brief overview of these key table-related HTML elements:

  • <table>: This is the main container element for the table. All the other table elements are nested within this tag.
  • <tr>: Stands for "table row" and is used to define a row in the table. It contains one or more <th> (table header) or <td> (table data/cell) elements.
  • <th>: Represents a header cell in the table. It is typically used in the first row or first column to label the data in that row or column. Text in <th> elements is usually bold and centered.
  • <td>: Represents a standard data cell in the table. It is used to contain the actual data that is being displayed. Text in <td> elements is typically not bold and left-aligned.
  • <caption>: An optional element that can be used to provide a title or a brief description for the table. It is placed immediately after the opening <table> tag.

Attributes for table

  1. border: Specifies the width of the border around the table. Common values are 0 (no border) or 1 (default).
  2. width: Sets the width of the table. This can be specified in pixels or percentage.
  3. cellspacing and cellpadding: Controls the space between cells and the padding within cells, respectively.
  4. colspan and rowspan: You can use the colspan and rowspan attributes to make a cell span multiple columns or rows.
